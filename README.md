# AFOP Skin Color Editor
Edit player skin color files (`.blueitemtype`) for Avatar: Frontiers of Pandora.

## How to Use

1. **Load files** — click **Load Folder** and browse to your skin color folder, or use **Load Files** to pick individual files.

2. **Edit a color** — click any color swatch to open the color picker.
   - Drag the **gradient square** to set saturation/brightness.
   - Drag the **rainbow bar** to shift hue.
   - Type a **hex code** directly or adjust the **R G B** spinboxes.

3. **Save** — click **Save** on a single row, or **Save All** to write every modified file at once.

> Each file has 4 color slots (Color 1–4). Only the hex color values are changed — everything else in the file is left byte-for-byte identical.
